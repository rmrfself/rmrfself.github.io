刺绣格式(dst)与svg格式转换

分类: JVM 发布于: 2014-04-12 20:12

部署方式笔记

安装libembroidery-convert

安装必须的编译环境

sudo apt-get install git build-essential qt4-dev-tools libqt4-opengl-dev kdelibs5-dev valgrind

克隆仓库

git clone https://github.com/Embroidermodder/Embroidermodder.git

进入子模块，执行编译过程

cd libembroidery-convert
qmake && make && make install
安装 apt-get install poppler-utils
apt-get install poppler-utils
